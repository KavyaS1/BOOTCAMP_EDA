Overview:

In this study, we perform exploratory data analysis (EDA) on loan application data to identify patterns related to default risk, financing behavior, and application outcomes.

Key Findings:
- Education & Default Risk: Lower levels of education are associated with a higher likelihood of default.

- Income & Financing Structure: Financing structure and approval ratios show minimal variation across income levels, suggesting borrower behavior plays a larger role than income constraints.

- Application Start Times and Approval: Applications started on weekends, particularly Saturdays, and those begun between 10am–4pm are more likely to be approved. More applicants cancel applications than wait for rejection.
